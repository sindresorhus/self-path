# self-path

> Get the path to this package - Useful for testing

You could spawn this CLI and check the path to assert certain things.

## Install

```
$ npm install --global self-path
```

## Usage

```
$ self-path
/Users/sindresorhus/n/lib/node_modules/self-path
```
